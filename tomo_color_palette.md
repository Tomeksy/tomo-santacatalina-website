# Tomo Color Palette

## Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| Tomo Red | `#DA240E` | Primary brand color, logo, CTAs |
| Tomo Green | `#48BB78` | Accent, health/nature elements |
| Cream | `#FFF8F0` | Backgrounds, soft contrast |
| Dark | `#111827` | Primary text |
| Gray | `#374151` | Secondary text |
| White | `#FFFFFF` | Backgrounds, contrast |
